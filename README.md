# Как Привезли? — Telegram Mini App

Мини-приложение для Telegram-бота магазина деликатесов. Интерфейс повторяет оригинальный каталог: тёмная тема, категории, корзина и оформление заказа.

## Возможности

- **Каталог** — поиск, категории, сетка товаров (5 позиций)
- **Категории** — отдельная страница с баннером и товарами категории
- **Карточка товара** — описание, счётчик количества, избранное
- **Корзина** — форма покупателя, промокод, блок «Вы смотрели»
- **Оформление заказа** — экран подтверждения со статусом и деталями
- **Профиль** — заказы, избранное, акции и новости
